## 0. 한 줄 요약

Cryptalk의 기존 **오프체인 포인트 시스템(`PointLog`)을 온체인 토큰 $TALK로 승격**시켜, "시세를 보러 왔다가 글을 읽고, 글을 읽으러 왔다가 시세를 본다"는 코인판형 순환 구조에 **경제적 보상·소각 루프**를 더한다. 핵심은 *기여(글·댓글·분석·추천)로 벌고, 혜택(프리미엄·노출·거버넌스)으로 쓰며, 쓰는 만큼 태운다*.

---

## 1. 설계 배경 — 왜 토큰인가

`coinpan-site-analysis.md` 7장에서 정리한 코인판의 리텐션 장치는 모두 **포인트(비화폐) 기반**이다.

| 코인판 장치 | 한계 | $TALK로 해결 |
|------------|------|-------------|
| 활동 포인트 → 레벨 | 사이트 밖에서 가치 0, 이탈 시 소멸 | 자산화 → 이탈 비용 발생, 외부 유동성 |
| 출석체크 포인트 | 단순 반복, 보상 매력 낮음 | 토큰 일일 보상 + 스트릭 가중 |
| 손익인증/추천 문화 | 명예 보상에 그침 | 추천이 보상 분배(큐레이션 마이닝)에 직접 연결 |
| 비회원 제한(가입 퍼널) | 가입 후 동기 약화 | 보유/스테이킹 기반 등급 혜택으로 지속 동기 |

현재 스키마(`PointLog.action = post | comment | attendance`)는 이미 **"행동 → 적립" 원장**이다. $TALK는 이 원장을 그대로 **발행 근거(emission ledger)**로 재사용한다. 즉 토크노믹스는 신규 시스템이 아니라 **기존 포인트 회계의 화폐화**다.

## 2. 토큰 개요

| 항목 | 값 |
|------|-----|
| 명칭 / 티커 | Cryptalk Token / **$TALK** |
| 발행 주체 | **라오스 현지 법인** (디지털자산 파일럿/라이선스 보유 또는 라이선스 파트너 제휴) |
| 표준 | ERC-20 (L2 우선: Base 또는 Arbitrum — 가스비·접근성·글로벌 유동성 고려) |
| 총 발행량 | **1,000,000,000 TALK (10억, 하드캡 고정)** |
| 인플레이션 | 추가 발행 없음(non-inflationary). 모든 보상은 사전 할당된 "커뮤니티 보상 풀"에서만 지급 |
| 소수점 | 18 decimals |
| 초기 비유통 처리 | 컨트랙트 vesting / 타임락으로 잠금, 온체인 공개 |
| 회계 통화 | 내부 가치 표기는 **USD 기준**(LAK 변동성·환관리 회피, 10장 참조) |

> **왜 고정 캡인가:** 무한 발행형 리워드 토큰은 리테일에서 "찍어내는 코인" 인식이 강해 신뢰를 잃는다. 고정 캡 + 소각으로 **디플레이션 압력**을 명시한다.

> **왜 라오스 발행인가:** 라오스는 2021년부터 정부 주도 **디지털자산(암호화폐 채굴·거래) 파일럿**을 운용하며 라이선스 기반 발행·거래 경로를 제공한다. 한국 대비 토큰 발행·거래소 라이선스에 대한 명문 경로가 있어 **발행 주체를 라오스 현지 법인으로 두는 것**이 규제 명확성 측면에서 유리하다(단, 실제 적격성·범위는 BOL/MTC 사전 확인 필수 — 10장).

## 4. 발행(Emission) — 어떻게 버는가

### 4.1 커뮤니티 보상 풀 방출 스케줄

400M을 **반감기형 5년 점감**으로 방출. 초기 활동 보상을 두텁게 하되 장기 희석을 방지.

| 연차 | 방출 비중 | 방출량(TALK) | 비고 |
|-----|----------|-------------|------|
| Year 1 | 35% | 140,000,000 | 부트스트랩(가장 두터움) |
| Year 2 | 25% | 100,000,000 | |
| Year 3 | 20% | 80,000,000 | |
| Year 4 | 12% | 48,000,000 | |
| Year 5 | 8% | 32,000,000 | 이후 트레저리 재원으로 연장 |

연차 풀은 다시 **주(week) 단위 예산**으로 쪼개진다. 한 주의 보상 풀은 그 주에 발생한 전체 기여 점수 비례로 분배된다 → **고정 예산 / 변동 단가** 구조(=인플레이션 방어).

### 4.2 행동별 기여 점수 (현 `PointLog`/`Vote` 매핑)

현재 적립 행동을 **기여 점수(weight)**로 환산하고, 주간 풀을 점수 비례 분배한다.

| 행동 | 현 스키마 근거 | 기본 weight | 보상 트리거 |
|------|---------------|------------|------------|
| 게시글 작성 | `PointLog.action="post"` | 10 | 작성 + 최소 체류/조회 충족 |
| 댓글 작성 | `PointLog.action="comment"` | 3 | 작성 |
| 출석체크 | `PointLog.action="attendance"`, `day` UNIQUE | 5 (스트릭 가중) | 1일 1회(KST), 연속일 보너스 |
| 받은 추천 | `Vote.value=+1`, `Post.upvotes` | 8 / 순추천 | **큐레이션 마이닝**(아래) |
| 한 추천(투표 행위) | `Vote` | 1 | 추천한 글이 상위로 가면 추가 보상 |
| 검증된 분석글 | `Post.priceAtPost` 존재 + 사후 적중 | 30 | 5장 9.2 "예측 검증" 연계 ★ |

> **검증된 분석글 보너스**가 핵심 차별점이다. 스키마에 이미 `priceAtPost`/`priceSymbol`이 있으므로, 작성 시점 가격 대비 이후 실제 가격으로 적중도를 계산해 **맞춘 분석가에게 가중 보상**한다. 코인판의 "유저 자유 기고 품질 편차" 문제(9.2절)를 경제적 인센티브로 교정.

### 4.3 큐레이션 마이닝 (추천 = 채굴)

- 글 보상의 일부를 **그 글을 일찍 추천한 유저들에게 분배**한다(`Vote` 타임스탬프 기준).
- 효과: 추천이 단순 명예가 아니라 "좋은 글을 먼저 발굴하면 보상" → 코인판 HOT게시판/핫코멘트 문화(6.3절)를 보상 루프로 전환.
- 어뷰징 방지는 6장 참조.

### 4.4 마이그레이션 보너스 (포인트 → 토큰)

TGE 시점의 기존 `User.points` 잔액을 스냅샷해 **에어드랍 풀(5%)에서 환산 지급**. "기존 활동이 자산이 된다"는 메시지로 코인판형 헤비유저 이주를 유도. (배포·청구는 라오스 법인 발행 컨트랙트 기준, 10.2 국경 간 판매 검토 반영)

## 5. 효용(Utility) — 어디에 쓰는가

토큰은 **반드시 쓸 데가 있어야** 발행 압력을 흡수한다. 사용처는 곧 소각·재순환 경로다.

### 5.1 소비형(소각 발생)

| 효용 | 설명 | 토큰 흐름 |
|------|------|----------|
| **글 상단 노출/부스트** | 코인판 AD 고정(8장)의 유저판. 자기 글/분석을 일정시간 상단 고정 | 50% 소각 / 50% 트레저리 |
| **프리미엄 분석 열람** | 공식 시장 브리핑(9.2절) 일부를 토큰 페이월 | 50% 소각 / 50% 작성자 |
| **닉네임/프로필 꾸미기, 배지** | 코스메틱 | 70% 소각 |
| **비회원·저레벨 제한 해제권** | 댓글 즉시열람 등(6.3절 장치) 구매 | 100% 소각 |
| **신고/검증 예치금** | 분석 신뢰도 챌린지 시 스테이크, 패배 시 소각·승자 분배 | 조건부 소각 |

### 5.2 보유/스테이킹형(잠금 → 유통 축소)

| 효용 | 설명 |
|------|------|
| **등급(Tier) 혜택** | 보유/스테이킹량에 따라 광고 제거, 수수료 할인, 분석 무료열람, 큐레이션 보상 가중 |
| **거버넌스 투표권** | 게시판 신설/삭제, 보상 weight 조정, 트레저리 집행 (veToken식 잠금가중 가능) |
| **분석가 스테이킹** | 분석글 작성 시 토큰 예치 → 적중 시 보상↑, 틀리면 슬래시(품질 본드) |
| **포트폴리오 프리미엄** | `PortfolioItem` 기반 고급 분석/알림을 스테이커에게 개방 |

### 5.3 거버넌스 범위 (DAO)

트레저리(15%)와 보상 파라미터를 점진적으로 커뮤니티에 이양:
1. 행동별 weight(4.2) 조정
2. 신규 코인 포럼/게시판 상장 제안·투표
3. 그랜트(분석가·번역·개발) 집행
4. 바이백&소각 비율 결정

## 6. 어뷰징 방지 (Sybil / 농사 방지)

채굴형 커뮤니티 토큰의 최대 리스크. 보상 원장이 곧 공격면이다.

| 위협 | 방어 |
|------|------|
| 다계정 출석/도배 | 보상 자격에 **stake/보유 하한**, 가입 시 인증(이메일+소셜, 추후 onchain reputation) |
| 셀프/품앗이 추천 | 동일 IP·기기·상호추천 그래프 탐지, 추천 보상에 **추천자 다양성 가중** |
| 저품질 양산 글 | weight에 **조회 체류·고유 추천자 수** 반영, 순추천(up−down) 음수면 0 |
| 봇 댓글 | 댓글 weight 낮음(3) + 속도 제한 + 반복 텍스트 패널티 |
| 분석 찍기 | 분석 보너스는 **본드 스테이킹 + 사후 검증** 통과분만 지급 |
| 추천 매수 | 보상은 즉시 지급이 아니라 **에포크 정산**(주 단위), 정산 전 어뷰징 무효화 가능 |

> 보상은 **클레임형(주간 정산 후 사용자가 청구)**으로 설계해, 정산 기간 동안 부정 적립을 회수할 수 있게 한다.

## 7. 소각 & 가치 환류 (Sink / Value Accrual)

고정 캡이므로 **순환·소각 설계가 가치의 핵심**이다.

1. **사용처 소각**(5.1) — 부스트/코스메틱/해제권의 50~100%를 영구 소각.
2. **수익 바이백&번**(burn) — 광고·제휴(코인판 8장 수익모델)·프리미엄 매출의 일정 비율로 시장에서 $TALK 매입 후 소각.
3. **스테이킹 잠금** — 등급·거버넌스·분석 본드로 유통량을 묶어 *유효 유통량* 축소.
4. **미클레임 만료** — 일정 기간 미청구 보상은 풀로 환수(롱테일 인플레 방지).

순효과: *활동↑ → 사용처 매출↑ → 소각↑ → 희소성↑*. 보상 방출(4.1)이 점감하는 5년차 이후 소각이 방출을 추월하도록 파라미터를 잡는 것이 목표(net-deflationary).

> **권고:** Phase 1에서 **오프체인으로 경제(발행/소각/물가)를 충분히 튜닝한 뒤** TGE. 토큰을 먼저 띄우고 경제를 맞추면 되돌릴 수 없다.

## 10. 리스크 & 규제 메모 (라오스 기준)

> 본 절은 일반 공개 정보 기반 가이드다. **실제 적격성·라이선스 범위·세율은 라오스 중앙은행(BOL)·기술통신부(MTC) 및 현지 법률자문으로 사전 확정**해야 한다.

### 10.1 라오스 디지털자산 라이선스 체계
- 라오스는 2021년부터 정부 주도 **디지털자산(암호화폐 채굴·거래) 파일럿**을 운용, BOL·MTC가 감독하고 소수 라이선스 사업자에게 채굴·거래소 운영을 허가해 왔다.
- **토큰 발행/거래소 연계**는 파일럿 라이선스 범위 안에서만 합법적 경로가 열린다. → ① 자체 라이선스 취득, 또는 ② **라이선스 보유 현지 파트너와 제휴**(발행·수탁·거래 창구)하는 두 갈래를 병행 검토.
- 라이선스 조건(자본금, KYC/AML 의무, 정부 분담금·수수료, 보고 의무)을 토크노믹스 운영비(트레저리 15%)에 반영.

### 10.2 토큰 성격 정의(증권성 회피)
- $TALK는 **"이익 배당"이 아니라 유틸리티·소각 토큰**으로 일관되게 프레이밍. 페이월/부스트/해제권 등 *명확한 소비 효용*(5.1)을 전면에.
- 스테이킹 "수익"은 배당이 아니라 **활동 보상 풀 분배·등급 혜택**으로 설계·표현(6장 정산 구조).
- 한국 사용자 대상 마케팅이라도 발행·판매 주체는 라오스 법인이므로, **한국 거주자 대상 공모/판매가 한국 규제(가상자산이용자보호법 등) 트리거가 되지 않도록** 분리 검토 필요(국경 간 판매 리스크).

### 10.3 외환·자금흐름(LAK 변동성·자본통제)
- 라오스 키프(LAK)는 변동성·평가절하 이력이 크고 외환 관리가 존재 → **내부 회계·효용 가격은 USD(또는 USDC) 기준**으로 운영, LAK 익스포저 최소화.
- 매출(광고·제휴·프리미엄) 정산, 바이백&번(7장) 재원의 통화·송금 경로를 라이선스·외환 규정에 맞게 설계. 스테이블코인 정산 채널 우선.

### 10.4 인프라·운영
- **전력/채굴 의존 아님:** 본 프로젝트는 PoW 채굴이 아니라 L2 기반 발행이므로, 라오스 수력발전·채굴 전력 정책(가뭄기 채굴 제한 등) 리스크와 무관. "채굴 코인"으로 오해되지 않게 커뮤니케이션.
- **현지 운영 리스크:** 인터넷·결제·인력 인프라, 규정 변경 빈도를 감안해 핵심 인프라(체인·DEX·수탁)는 글로벌 L2/스테이블코인 위에 두고, 라오스는 **법인·라이선스·정산 레이어**로 활용.

### 10.5 공통
- **거래소 상장/유동성:** 초기엔 DEX(L2) 중심, 가두리 방지 위해 유동성 15% 충분히 배정. 라이선스 범위 내 현지 거래 창구는 파트너 경유.
- **가격 변동성 → UX 악화:** 부스트/열람 가격을 토큰 고정수량이 아니라 **USD 페그(시점 환산)**로 책정해 변동성으로 효용이 깨지지 않게.
- **보상 인플레 체감:** 4.1 점감 + 7장 소각이 핵심. 정기적으로 "방출 vs 소각" 대시보드 공개(투명성 = 신뢰).
